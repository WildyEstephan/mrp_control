MRP Control QTY Production by Employee

Este modulo es para controlar las cantidades para cuando necesitas saber cuantos items manufacturaron tus empleados.



1) Delivery order list view

Here user can see the field named 'Number of Days' from where user can see how many days ago the delivery order has been created.

Navigation: Inventory -> Transfer

2) Delivery orders search view

Here users can search for any number of days for get the listed delivery Order for that day, enter number and select "Number of Day" in list of search field.

Navigation: Inventory -> Transfer

3) Delivery order list view

Here you can see, delivery order searched by number of days.
